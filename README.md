# 🌦️ Weather Data Analysis and Prediction

## 📌 Project Overview

This project analyzes historical weather data and predicts future temperature trends using **Linear Regression**, one of the fundamental machine learning algorithms for regression tasks. The project demonstrates data preprocessing, visualization, model training, evaluation, and future forecasting using Python.

---

## 🎯 Objective

- Analyze historical weather data.
- Identify temperature trends over time.
- Train a Linear Regression model.
- Predict future temperatures.
- Visualize historical and predicted temperature trends.

---

## 📂 Dataset

**Dataset:** Daily Delhi Climate Data

The dataset contains daily weather observations including:

- Date
- Mean Temperature
- Humidity
- Wind Speed
- Mean Pressure

Source:
https://www.kaggle.com/datasets/sumanthvrao/daily-climate-time-series-data

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📚 Machine Learning Algorithm

**Linear Regression**

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values. In this project, it is used to forecast future temperatures based on historical weather data.

---

## 📊 Project Workflow

1. Import required libraries
2. Load the weather dataset
3. Explore the dataset
4. Check for missing values
5. Convert date into numerical format
6. Split the dataset into training and testing sets
7. Train the Linear Regression model
8. Predict temperatures
9. Evaluate the model using Mean Squared Error (MSE)
10. Visualize Actual vs Predicted temperatures
11. Forecast future temperatures
12. Visualize future temperature trends

---

## 📈 Results

The model successfully:

- Learned historical temperature patterns
- Predicted future temperatures
- Generated visualizations for analysis
- Demonstrated basic time-series forecasting using Linear Regression
